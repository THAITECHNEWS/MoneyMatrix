**Hub Pages (Loan Type Pages):**

- `/personal-loans`
- `/installment-loans`
- etc.

**Spoke Pages (Location Pages):**
- `/locations/payday-loans-in-los-angeles-ca`
- `/locations/payday-loans-in-chicago-il`
- etc.

### Linking Structure:

1. **Hub ‚Üí Spokes:**
   - Loan type pages link to top 10-20 location pages

2. **Spoke ‚Üí Hub:**
   - Location pages link back to loan type page

3. **Spoke ‚Üí Spoke:**

   - "Popular Locations" section

4. **Content ‚Üí Location Pages:**
   - Articles mention and link to relevant location pages

### Internal Link Distribution:

- **Each loan type page:** Links to 20-30 location pages
- **Homepage:** Links to top 10-15 location pages

## ‚ö†Ô∏è Common Pitfalls to Avoid

1. **Duplicate Content**
   - Don't copy-paste content between cities
   - Make each page unique
   - Use city-specific data

2. **Keyword Stuffing**
   - Natural keyword usage
   - Focus on user value
   - Don't over-optimize

3. **Thin Content**
   - Minimum 1,500 words per page
   - Substantial, valuable content
   - Not just store listings

4. **Poor Internal Linking**
   - Don't link to every page from homepage
   - Use hub & spoke model
   - Natural link placement

5. **Ignoring User Experience**
   - Fast load times
   - Mobile-friendly
   - Easy navigation
   - Working store locator

6. **Not Tracking Performance**
   - Set up analytics
   - Monitor rankings
   - Track conversions
   - Adjust strategy based on data
