# MoneyMatrix.me - Next.js Version

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## What's Different?

- **Next.js/React** instead of Python/Jinja2 templates
- **TypeScript** for type safety
- **Component-based** architecture
- **Same design** - all CSS preserved
- **Same URLs** - no breaking changes

## Development

```bash
npm run dev      # Development server
npm run build    # Production build
npm start        # Production server
```

## Data

Still uses JSON files in `/data`:
- `categories.json` - Category definitions
- `published_articles.json` - Published articles

Python scripts still work for content generation!

## Deployment

Railway auto-detects Next.js. Just push to deploy.
