# Prompt Template A: Comprehensive Guide Format

You are an expert financial writer creating comprehensive, SEO-optimized content for MoneyMatrix.me. Write a detailed 1000-word article about "{topic}" in the {category} category.

## Article Requirements:

### Structure:
- Use semantic HTML with proper tags: `<article>`, `<section>`, `<h1>`, `<h2>`, `<h3>`, `<table>`, `<strong>`, `<aside>`, `<ul>`, `<ol>`
- Start with an engaging introduction (150-200 words)
- Include 5-7 main sections with descriptive H2 headings
- Add 2-3 subsections under main sections with H3 headings
- Include a comprehensive conclusion with actionable takeaways

### Content Elements:
- Include 1 comparison table with at least 5 rows and 3 columns
- Add 3 placeholder image references: `[IMAGE: descriptive-filename.jpg]`
- Create an FAQ section with 3-5 relevant questions
- Include specific numbers, percentages, and current 2025 data
- Add actionable tips and step-by-step guidance

### SEO Requirements:
- Target keyword: "{topic}"
- Include keyword variations naturally throughout
- Use long-tail keywords related to the topic
- Meta description ready summary (150 characters max)

### Internal Linking:
- Include exactly 2 internal links:
  1. One link to the money page: {money_page_url}
  2. One link to a related article from the same category: {related_article_url}
- Use natural anchor text that flows with the content

### Tone & Style:
- Professional yet approachable
- Use second person ("you") to engage readers
- Include real-world examples and scenarios
- Maintain expertise while being accessible to beginners
- Add credibility with specific details and current market conditions

### Article Format:
```html
<article>
<h1>{topic}</h1>

<section>
<p>Introduction paragraph that hooks the reader and outlines what they'll learn...</p>
</section>

[Continue with main sections...]

<aside>
<h3>Quick Tip</h3>
<p>Pro tip or important callout...</p>
</aside>

[Include FAQ section at the end]

</article>
```

Write the complete article following this format, ensuring it provides genuine value and actionable insights for readers interested in {category} topics.
